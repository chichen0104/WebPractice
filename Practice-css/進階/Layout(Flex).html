CSS Flex 可以說是近年最成熟且被廣泛運用的技巧。圍繞著 display 和 flex 的設定，改善 Inline-Block 在上下、 左右對齊設定上的缺陷，提供專屬、整套的解決方案如下：

核心 display: flex
核心 flex: none|auto
左右對齊 justify-content: flex-start|center|flex-end
上下對齊 align-items: flex-start|center|flex-end
